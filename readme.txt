# ShotMaster for Blender

Version 1.4
Author: Zac Counts

## Description
ShotMaster is an advanced camera management addon for Blender designed for film production and previsualization workflows. It provides powerful tools to organize, manage, and render multiple cameras with custom settings.

## Features
- Create and manage multiple cameras with unique settings
- Organize cameras into groups for better workflow
- Set camera-specific render settings and output paths
- Customize camera properties like sensor size and white balance
- Support for depth of field with focus target objects
- Define render passes (beauty, diffuse, shadows, etc.)
- Track shot information (shot size, type, equipment)
- Batch render cameras with different settings
- Quick viewport positioning tools

## Installation
1. Download the `shotmaster_1_4.py` file
2. Open Blender and go to Edit > Preferences > Add-ons
3. Click "Install..." and select the downloaded file
4. Enable the addon by checking the box next to "3D View: ShotMaster"

## Location
The ShotMaster panel can be found in the 3D View sidebar (press N to open) under the "ShotMaster" tab.

## Usage

### Master Settings
- Set global output path and frame range
- Configure default resolution and render engine settings

### Camera Groups
- Create camera groups to organize shots by scene, sequence, etc.
- Customize group settings for output path, render engine, etc.
- Groups can be collapsed/expanded for better workspace organization

### Camera Management
- Create, duplicate, and delete cameras
- Assign cameras to groups
- Set camera-specific properties like lens, sensor size, DOF
- Add detailed information like shot size, type, and equipment

### Rendering
- Render stills or animations from individual cameras
- Batch render all cameras or specific groups
- Use custom render passes for compositing workflows

## Requirements
- Blender 4.3.2 or higher
